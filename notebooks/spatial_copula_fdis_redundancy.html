Rank data

# https://stats.stackexchange.com/questions/355085/when-modeling-a-copula-you-need
# -to-generate-pseudo-observations-why-what-is

# Copula models based on pseudo-observation (normalized ranked data), not on the
# original dataset. That due to the Sklar's theorem (the backbone of the copula
# model). From Sklar's theorem, copula is a function of uniform margins. Hence,
# you need to transform the margins of your dataset to the standard uniform margins,
# in order to obtain copula data. For RVineCopula, if you simulate your data from
# RVineSim, then you do not need to transform the margins, however, if your data
# is a real data, then make sure that you transform it to the uniform distribution
# using pobs function.

data_copula_redundancy$fitted_redundancy <- pobs(data_copula_redundancy$fitted_redundancy)

hist(data_copula_redundancy$fitted_redundancy)
